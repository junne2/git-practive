git clone _해당 저장소 git url
깃 파일을 가져와서 작업 할수 있도록 세팅 해준다.

$ git init
# 현재 프로젝트에서 변경사항 추적(버전 관리)을 시작

$ git add index.html
# 변경사항을 추적할 특정 파일(index.html)을 지정

git status

git add . # 모든 파일의 변경사항을 추적하도록 지정

git status

$ git commit -m '프로젝트 생성'
# 메시지(-m)와 함께 버전을 생성.

git commit -m 'Start project'
